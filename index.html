Subir a GitHub

Para subir este proyecto a GitHub Pages y compartirlo:

  1. Crea un nuevo repositorio en GitHub
  2. Sube todos los archivos (HTML, modelos 3D, etc.)
  3. Ve a la configuración del repositorio
  4. Habilita GitHub Pages en la sección correspondiente
  5. Selecciona la rama principal (main/master)
  6. ¡Tu sitio estará disponible en https://tuusuario.github.io/turepositorio/

Estructura del Proyecto

La estructura de archivos para tu proyecto debe ser:

/ (raíz del repositorio)
│
├── index.html (este archivo)
├── models/
│   ├── Cartel_acri_fiesta.glb
│   └── Cartel_acri_fiesta.usdz
├── images/ (opcional)
└── README.md (documentación)

Compatible con Móviles

Funciona perfectamente en dispositivos iOS y Android gracias a Model Viewer.

Realidad Aumentada

Permite visualizar el modelo 3D en tu espacio mediante AR.

Modelos 3D

Soporta formatos GLB para dispositivos generales y USDZ para iOS.

Responsive

Se adapta a diferentes tamaños de pantalla, desde móviles hasta escritorio.
